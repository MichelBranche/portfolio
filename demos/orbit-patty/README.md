# Orbit Patty Aosta

Sito statico (HTML + CSS + JS) per un fast food a tema spazio con burger, fries, combo e sezione contatti.

## Cosa include
- Header fisso con menu mobile
- Tema light/dark (salvato in localStorage)
- Menu dinamico con filtri e ricerca
- Modal prodotto e modal delivery
- Layout responsive

## Avvio rapido
Apri `index.html` con Live Server (VS Code) oppure con qualsiasi server statico.

## File principali
- `index.html` contenuti e sezioni
- `styles.css` stile e responsive
- `script.js` dati menu (array `ITEMS`) e logica filtri/modals

## Personalizzazioni
- Aggiorna indirizzo, orari e contatti nella sezione "Dove atterriamo"
- Cambia prodotti, descrizioni e prezzi in `script.js`
- Regola palette e bordi in `:root` dentro `styles.css`
